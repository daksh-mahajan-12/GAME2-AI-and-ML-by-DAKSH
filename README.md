# GAME2-AI-and-ML-by-DAKSH
This project is a Python-based implementation of the classic "Stone, Paper, Scissors" game, where a user competes against the computer. The game uses Python’s random module to simulate the computer’s choice and simple conditional logic to determine the winner.
Key Features:
- User vs. Computer gameplay
- Randomized computer choices
- Score tracking and replay option
- Simple, modular code structure
- Optional GUI for enhanced interaction
Technologies Used:
- Python 3.x
- random module
- (Optional) tkinter for GUI
